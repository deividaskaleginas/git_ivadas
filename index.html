Terminalo komandos

  • ls - išvardina aplankalus ir failus esamoje direktorijoje
  • cd - (change directory) keičia direktoriją (vaikšto tarp aplankalų)
  • TAB - skirtas užpildyti terminalo komandą arba pateikti pasiūlymus
  • CTRL C - atšaukia vykdomą veiksmą / nutraukia komandą
  • Vidurinis/dešinys pelės mygtukas - įklijuoja tekstą į komandinę eilutę
  • ARROW UP/DOWN - vaikščiojimas per terminalo komandų istoriją
  • cd .. - grižta aplankalu atgal

Git Komandos

  • git init - inicializuoja git repozitorija
  • git status - nurodo repozitorijos statusą
  • git add ./*/file- pridėti failus prie paruoštų comitinimui
  • git commit -m 'žinutė' - sukuriama kodo versija su žinute
  • git restore ./*/file- panaikina nekomitintus pakeitimus
  • git restore --stage ./*/file- isima nurodyta faila/us is staged zonos.
  • git push - nusiuncia naujus logalius commit'us i serveri
  • git log - rodo git'u istorija
